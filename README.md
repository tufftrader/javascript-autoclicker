# JavaScript Auto Clicker

A fast auto clicker that works on any website via browser console.

## How to Use

1. **Open any website**
2. **Press F12 or Inspect** to open browser console  
3. **Paste the code** into the console
4. **Press Enter**
 
**Start/Stop:**
- Press **Ctrl + Shift + C**

The auto clicker will click wherever your mouse cursor is positioned.

## Speed

Clicks at **1000 clicks per second** (1ms interval).

}, 1); // Change 1 to higher number for slower clicks

## Notes

- Works on any website
- Clicks at mouse cursor position
- Some websites may detect/block rapid clicking
- Close console or refresh page to fully stop
